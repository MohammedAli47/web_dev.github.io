What's the basic salary for a Jr. Web developer?

In the United States, the average salary is more than 75k$ annually (According to Indeed.com).

In the United Kingdom, the average salary is more than 65k$ annually (According to Quora.com).

But for senior developers with an experience over three years, it goes up to 110k$ per year and higher!!

What is Web development?

Anything on the internet is made with web development using programming languages like HTML, CSS, Javascript and other programming languages used like Ruby, Python, Java and others that are very important for adding extra functionalities to your website.

What jobs are available with Web development?

1-Front-end developer : Someone who deals with what the user/visitor of the website sees like buttons, images, text and all these things using HTML, CSS and Javascript.

2-Back-end developer : Someone who deals with what the user/visitor doesn't see like processing data after clicking a button using Javascript, storing the data using SQL, Machine learning using Python and many other things.

3-Full stack developer : A mix of the two, but not as skilled as any of them, he uses HTML, CSS, Javascript, A database language(like SQL), an Object Object-Oriented language (like Python) and a framework (like React) to build a complete web app.

Full stack web development is the most used and needed because it reduces the number of developers working and the time needed to build the app/site. The word "stack" refers to the technology used beside HTML, CSS and Javascript.

Examples of available stacks and their differences :

1-The LAMP stack : Uses Linux as its operating system, Apache as the Web server, MySQL for database management and PHP as the Object Oriented scripting language. (Sometimes Perl or Python is used instead of PHP.)

2-The MEAN or MERN or MEVN stack: This stack uses MongoDB for data management, Express.js is a modular web application framework package for Node.js, a framework and Node.js which is the runtime that the stack runs on.

The Angular framework is based on Typescript(A superset of Javascript) and was made by Google.

The React framework is based on normal Javascript and was made by Facebook.

The Vue framework is based on HTML syntax that helps you to use Advanced Javascript techniques and it was made by a group called Evan You who worked for google and used Angular.

3-The Django stack : Based on Python, which is Object-Oriented and depends heavily on it to build full stack apps that use Machine learning, Data science and other things that python is superior at.

4-The Ruby on Rails stack : Based on Ruby, which is Object-Oriented and the Rails framework designed especially for it.

Important things to know:

1-Web page : A simple document displayable by a web browser.

2-Web site : A group of globally accessible, interlinked web pages which have a single domain name.

3-Web application : A software or program which is accessible using any web browser.

4-Web browser : A software application for accessing information on the World Wide Web.

5-Web server : A computer hosting one or more websites. "Hosting" means that all the web pages and their supporting files are available on that computer.

6-What is an HTTP and HTTPS request?

A request that your web browser makes to access the URL you wantto visit and the "S" means that the website is safe from unauthorized permission.
