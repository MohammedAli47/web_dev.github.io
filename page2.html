How to get started?

You can either take free courses(online websites, youtube videos,...etc) or paid courses, but you have to first understand the differences.

Comparison Free courses Paid courses
Information They teach you all you need to know about the topic but need to search on your own to get the full information. The same applies for paid courses, but taking free courses teaches you how to search about the info on your own.
To-Do Tasks Some have mini tasks or questions between lessons, others just offer raw info. They have multiple tasks between lessons.
Super-vision You learn on your own with out anyone following your progress. You are constantly being followed.
Projects No real projects for you to build at the end of the course. You have to build a project on your own before finishing.
Course certificates Some have, some don't. All of them have.

Where do you write the code?

Either on Text editors like a note pad, Code editors VS Code or IDE's(Integrated Development Environment) like Pycharm(for Python).

What's the difference?

Text editors : For languages that don't need compiling like HTML.

IDE's : For certain languages only.

Code Editors : A mix of the two and the best kind to write your code.

Best Code Editors to download :

1-Visual Studio Code (VS Code) : Made by Microsoft, it is one of the best code editors you'll find and it has a wide variety of extentions. Download link

A 6 month plan to get started with Web-development :

Months What to do
Month No.1 Learn the basics of HTML and CSS(from SoloLearn or freeCodeCamp).
Month No.2 Learn Javascript and how to problem solve with it.
Month No.3 Build some websites with your current skills.
Month No.4 Learn basic web design fundementals and tools(for front-end) and MongoDB, SQL or any language for databases(for back-end).
Month No.5 Learn a stack(Preferably MEAN or MERN).
Month No.6 Build a full stack web application and deploy it to the internet.

Important concepts to learn :

1-Variables // 2-Numerical operations // 3-Assigning values // 4-Comments // 5-Logic // 6-Loops // 7-Break and continue // 8-Functions // 9-Recursion // 10-Modules/Libraries // 11-Arrays // 12-Object Oriented Programming(OOP) // 13-Working with external files
